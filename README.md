# Template NextJS, TypeScript y TailwindCSS

Template creado a partir de `npx create-next-app --typescript`.

Incluye [TailwindCSS](https://tailwindcss.com/) y [Prettier](https://prettier.io/).

## Utilidades

- Tailwind:
  - TwAssemble
- Viewport size
  - viewportContext
  - viewportProvider
  - useViewport
